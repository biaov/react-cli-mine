## 使用

<h3 id="create">创建初始化 React 项目</h3>

* name 表示你的项目名称，是可选的。
* 之后就按照提示走就可以了。
* 注意：如果你没有写 name 的话，后面要写上。

```Basic
rcm create [name]
```

<h3 id="version">查看版本号</h3>

* 你可以使用 `-v` 或者 `--version` 选项。

```Basic
rcm -v
```

<h3 id="help">查看所有选项和命令</h3>

* 你可以使用 `-h` 或者 `--help` 选项。

```Basic
rcm -h
```
